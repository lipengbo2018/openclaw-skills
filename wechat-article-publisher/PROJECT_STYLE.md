# 破局圈友风写作风格指南

> 基于两篇高认可度文章提炼的写作规则，适用于复盘、经验分享类文章

---

## 1. 语气特点

| 规则 | 说明 | 示例 |
|------|------|------|
| 第一人称 | 全程用"我"叙述，不装B | "我很普通，这次也没有贵人相助" |
| 真诚坦率 | 敢于承认自己不足和失败 | "我能力不行"、"我是小白" |
| 亲切口语 | 像朋友聊天，不端着 | "废话不多说"、"讲讲我是怎么干的" |
| 归因社群 | 成功归功于教练/圈友/社群 | "感谢破局圈友的帮助" |
| 鼓励为主 | 给读者信心，强调"你也可以" | "我这样子的都出单，大家一定也可以" |

---

## 2. 句式特点

| 规则 | 说明 | 示例 |
|------|------|------|
| 长短结合 | 节奏感，不单调 | "我花了7天，跑了3个账号" |
| 善用排比 | 增强气势 | "时间利用效率低、发的视频不涨粉、维护2个账号应该是极限了" |
| 设问自答 | 引起共鸣 | "问题：一开始就掉队了，这可咱整？？？" |
| 强调重点 | 用"：**"标注 | "**我得认清现实，找适合自己的学习方法**" |
| 口语连接 | 用"然后"、"其实"、"但" | "其实5个账号都准备好了，我实际运营3个" |

---

## 3. 开头和结尾

### 开头公式
```
我是[身份/名字]，[时间]加入[社群]，[核心成果]

如果[读者痛点]，[接下来的文章或许会对你有帮助！]

[简要背景/身份定位]
```

### 开头示例
> 大家好，我是李老师otw，2023年5月加入破局，坚持没下牌桌，这次终于跑出来了。
>
> 如果你与我有同样的经历，又或者你还在迷茫期，**接下来的文章或许会对你有帮助！**
>
> 提前透露下，我很普通，这次也没有贵人相助。

### 结尾公式
```
[感谢] → [展望/号召] → [鼓励]
```

### 结尾示例
> 感谢大家的关注，我遇到的这种种失误，以及所付出的这么多努力，希望能给大家些参考。
>
> 最后，**大家拼的时候主要保重身体哈哈！**

## 4. 标题风格

| 规则 | 示例 |
|------|------|
| 数字具体 | 6天980GMV、4个月、10位 |
| 结果导向 | 实现收益、拿到结果、赚了多少 |
| 身份标签 | 普通人、零基础、纯种小白、小白 |
| 口语化 | 心理路程、踩坑复盘、实战总结 |

### 标题模板
```
[身份] + [时间/动作] + [结果]
普通人零基础入局AI漫剧，4个月带10位破局小白100%实现收益

[项目] + [复盘/经验] + [数字]
数字人视频号带货复盘：6天980GMV，纯种小白的心理路程
```

---

## 5. 文章结构

### 标准结构（时间线型）
```
# 一、先说下我在[社群]的经历
   - 按时间线列出关键节点

# 二、[第一阶段]：描述 + 问题
   - 具体做了什么
   - 遇到什么问题

# 三、[转折点/解决方案]
   - 怎么发现的转机
   - 做了什么改变

# 四、[第二阶段]：执行 + 结果
   - 调整后的行动
   - 取得的成果

# 五、[第三阶段]：持续/变现
   - 后续进展
   - 数据成果

# 六、复盘总结
   - 核心经验1-4条
   - 每条独立成段

# 七、[后续规划/尾声]
   - 接下来要做什么
   - 感谢 + 互动
```

### 关键节点配图
- 数据截图（成交、流量、粉丝）
- 成果展示（作品、收益）
- 日常记录（群聊截图、工作场景）

---

## 6. 写作检查清单

写完后对照检查：

- [ ] 开头有自我介绍+核心成果
- [ ] 强调"我是普通人/小白"，不装B
- [ ] 成功归因于社群/教练/圈友
- [ ] 有具体数据（时间、金额、人数）
- [ ] 失败经历真实不隐瞒
- [ ] 结尾有感谢+鼓励读者
- [ ] 标题带数字+结果
- [ ] 关键节点有截图

---

## 7. 禁用规则

| 禁止 | 原因 |
|------|------|
| 装B吹牛 | 失去读者信任 |
| 只讲成功不讲失败 | 没有参考价值 |
| 成功归因于自己 | 显得自私 |
| 过于专业化术语 | 脱离读者 |
| 负能量抱怨 | 传播焦虑 |

---

## 8. 快速模板

```
# [项目/赛道]复盘：[时间]，[结果]

大家好，我是[名字]，[什么时候]开始做[项目]。

## 一、我的情况
[简单介绍自己的背景，强调普通/小白]

## 二、经历
[按时间线讲述过程，最好有失败有成功]

## 三、关键转折
[发现了什么、做了什么改变]

## 四、数据结果
[具体数字，要真实]

## 五、核心经验
1. [经验1]
2. [经验2]
3. [经验3]

## 六、后续
[接下来要做什么]

感谢[社群/教练/圈友]，希望对大家有帮助！
```
